# ROADIE Voice Wave

Internal visual state component for Roady speech activity.

```tsx
import { VoiceWave, normalizeWaveEnergy } from '@/features/voice-wave'
import type { VoiceWaveProps } from '@/features/voice-wave'
```

Use `VoiceWave` as a compact visual indicator when ROADIE is speaking. Keep listening and thinking states as explicit text labels.
